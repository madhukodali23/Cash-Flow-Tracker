# Cash Flow Tracker

A simple and interactive web application to manage salary and track daily expenses in real time. This project is built using HTML, CSS, and Vanilla JavaScript without any frameworks to strengthen core JavaScript concepts.

---

## Features

* Add and display total salary
* Add multiple expenses with name and amount
* Automatic balance calculation
* Delete individual expenses
* Data persistence using localStorage
* Visual representation using Chart.js (Pie Chart)
* Real-time UI updates without page reload

---

## Project Objective

The goal of this project is to understand and implement:

* DOM Manipulation
* Event Handling
* Data Persistence using localStorage
* Real-time calculations in JavaScript
* Dynamic UI updates

This project simulates how real-world applications like finance trackers and dashboards work.

## How It Works

1. User enters salary
2. User adds expenses (name and amount)
3. Application calculates total expenses
4. Balance is updated automatically
5. Data is saved in browser using localStorage
6. Chart updates dynamically based on data

---

## How to Run

1. Download or clone the repository
2. Open index.html in your browser
3. Start adding salary and expenses
